# Monotone Name Generator Suite

A modern, minimalist, and high-contrast character and story creation dashboard. This suite provides specialized generators for classic names, fantasy characters, physical descriptions, and job categories, all wrapped in a sleek, permanent dark mode aesthetic.

## Features

- **Classic Name Generator**: Generates male and female names from a curated list of hundreds of names.
- **Fantasy Name Generator**: Specialized generator for fantasy-themed male and female names.
- **Physical Description Generator**: Provides detailed, random physical trait combinations.
- **Job Category Generator**: Generates job titles along with their categories and wealth potential levels.
- **Monotone Aesthetic**: A premium, unified dark mode design using high-contrast blacks, whites, and greys.
- **Mobile Responsive**: Fully optimized for various screen sizes with clean, fixed navigation.

## Technology Stack

- **HTML5**: Semantic structure for accessibility.
- **Vanilla CSS**: Custom design system using CSS variables and modern flex/grid layouts.
- **Vanilla JavaScript**: Lightweight, asynchronous logic for dynamic data fetching and generation.
- **JSON Data**: Powered by specialized datasets for names, traits, and jobs.

## How to Run

1.  Clone the repository:
    ```bash
    git clone https://github.com/grocerysushi/name-generator.git
    ```
2.  Open the project in your favorite code editor.
3.  Run a local live server (e.g., using VS Code's Live Server extension or `npx serve`).
    > [!NOTE]
    > Due to browser security policies, local data fetching (JSON) requires the files to be served via a local server rather than opened directly as `file://`.
4.  Navigate between the generators using the menu at the top.

## Project Structure

- `index.html` / `script.js`: Classic Name Generator.
- `fantasy.html` / `fantasy.js`: Fantasy Name Generator.
- `physical.html` / `physical.js`: Physical Description Generator.
- `jobs.html` / `jobs.js`: Job Category Generator.
- `style.css`: Unified design system.
- `*.json`: Generator datasets.
